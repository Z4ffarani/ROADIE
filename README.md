# PROJETO
A ROADIE é uma plataforma imersiva voltada ao upskilling e à transição profissional, que combina trilhas de aprendizado personalizadas com dados do mercado de trabalho. Em um universo virtual de constelações, cada estrela representa competências e requisitos alinhados às profissões emergentes. A solução constrói um perfil profissional dinâmico, conecta usuários a empresas e instituições de ensino e orienta a evolução de carreira com foco em adaptabilidade, propósito e relevância futura.

<br>

# INSTRUÇÕES
1. Em um terminal, clonar o repositório:
```bash
git clone https://github.com/Z4ffarani/ROADIE.git
```

2. No terminal, navegar até a pasta do projeto:
```bash
cd ROADIE/roadie
```

3. No terminal, instalar o pacote de módulos do Node.js:
```bash
npm install
```

4. Iniciar o servidor de desenvolvimento:
```bash
npm run dev
```

5. Acessar o site no endereço: **http://localhost:3000**

<br>

# FUNÇÕES
- **Experiência imersiva e intuitiva** | Interface visual inspirada no conceito de universo e constelações, projetada para orientar o usuário de forma clara, atrativa e didática ao longo da jornada profissional.

- **Exploração dinâmica de carreiras** | Navegação fluida entre trilhas, constelações e estrelas, permitindo ao usuário descobrir profissões emergentes, competências e caminhos de evolução de maneira progressiva e gamificada com quizzes.

- **Progresso e perfil do usuário** | Armazenamento temporário do progresso da jornada, histórico de aprendizado e currículo dinâmico no ambiente do navegador, simulando um cenário real de uso.

- **Dashboards interativos** | Empresas parceiras acessam áreas exclusivas que permitem analisar perfis de potenciais candidatos às vagas abertas e abrir novas conversas.

- **Acesso multiplataforma** | Design responsivo que garante uso consistente e acessível em desktops, tablets e smartphones, acompanhando o usuário em qualquer dispositivo.

- **Arquitetura modular e escalável** | Uso de componentes reutilizáveis que asseguram consistência visual, facilidade de manutenção e evolução contínua da plataforma.

<br>

# OBSERVAÇÕES
- O projeto trata-se de um protótipo desenvolvido exclusivamente em front-end, com foco em validação de conceito, experiência do usuário e visualização da jornada de upskilling.

- Conteúdos educacionais e cursos apresentados são fictícios e não auditados, utilizados apenas para demonstrar fluxos, progressão e integração conceitual.

- Arquitetura pensada para ser aberta a integrações externas, permitindo futura conexão com chatbots, assistentes baseados em IA e outros serviços inteligentes.

- Solução concebida com potencial de escalabilidade em nuvem, facilitando evolução para ambientes distribuídos e alta disponibilidade.

- Possibilidade de parcerias entre empresas e instituições de ensino, com proposta de uso de smart contracts para garantir transparência, rastreabilidade e confiança nas relações.

- Necessidade de evolução das camadas de segurança, especialmente no que diz respeito à manipulação de dados, autenticação e proteção contra acessos indevidos.

- Expansão disponível para integração com back-end dedicado, viabilizando persistência de dados, gestão de usuários e funcionalidades avançadas em ambiente de produção.
